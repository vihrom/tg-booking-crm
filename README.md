# Telegram-бот для онлайн-записи на услуги

Асинхронный бот на `aiogram 3` для автоматизации записи клиентов (например, в салон красоты, автосервис или к мастеру) с панелью управления для админа.

## Что умеет бот

### Для клиентов

* Пройти быструю регистрацию при первом старте (имя, телефон).
* Посмотреть список услуг и мастеров.
* Выбрать свободный день и время для записи.
* Посмотреть свои активные бронирования.
* Проверить баланс в программе лояльности.
* Написать в чат техподдержки (прямая связь с админом через бота).

### Для админа

* Добавлять и удалять услуги, менять цены и длительность.
* Добавлять мастеров и привязывать к ним услуги.
* Настраивать график работы сотрудников.
* Отвечать на сообщения пользователей из техподдержки прямо через интерфейс бота.
